安装

```
npm i vite-plugin-compression -D
```

引入

```
import viteCompression from 'vite-plugin-compression'
 
 plugins: [vue(), viteCompression()]
```

打包后就会生成 gzip 文件了，但是服务端 nginx 还需要配置一下才能生效

```
http {
    gzip_static on;
    gzip_proxied any;
}
```
