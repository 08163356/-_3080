匹配空行

```
^\s*\n
```

需求如下：

- 数据源：含有多个**空格、换行符**的多段文本
- 文本处理过程：识别出这些**空格、换行符**(替换为统一的单个换行符，或HTML的`<br />`标签)
- 数据持久化 --> 数据库

```
[\ \r\n]+
```
