在线检测：

https://regexr.com/

网址匹配：

```
/^(?:http(s)?:\/\/)?([A-Za-z0-9]-?)+(?:\.[\w\.-]+)+[\w\-\._~:/?#[\]@!\$%&\*\+,;=.]+$/
```

这个正则表达式作用是什么？

```
/^.*[*\.\-=,<>:()\][&^%$#@|!~_?]{2,}.*$/
```
<!-- more -->

```
解释：匹配输入内容从开始到结尾的所有内容中出现过任意*\.\-=,<>:()\][&^%$#@|!~_?两个以上的字符
```
