目的：

当前有一个bug需要复现浏览器全屏和半屏卡死的问题

方案

使用bat脚本，由于播放的是youtube视频，发现全屏只需要发送快捷键f就可以了

```bat
@echo off
set x=1
:onceAgain
adb shell input keyevent 34
timeout /t 5
%x% = %x% + 1
echo "执行了%x%次"
goto onceAgain
```
