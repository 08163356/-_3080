1. ```
   git log filename
   ```

   

可以看到fileName相关的commit记录

2. ```
   git log -p filenam
   ```

   

可以显示每次提交的diff

3. 只看某次提交中的某个文件变化，可以直接加上fileName

```
git show c5e69804bbd9725b5dece57f8cbece4a96b9f80b filename
```



4.查看某次commit的改动

```
git diff COMMIT^!
```

5.查看修改的详细文件列表

```
git show 9ddc9dca00b --stat
```
