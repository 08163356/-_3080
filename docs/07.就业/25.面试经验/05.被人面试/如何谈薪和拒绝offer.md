## 谈薪

### 1.讲个人优势

```
过往：
1.稳定性高
2.过去也取得了很多成绩
3.过去的薪资情况，也希望一定的涨幅。如果降或者平进入那么稳定性上是否值得考量？
4.过往公司五险一金都是顶格，奖金福利什么的也挺高
5.有学历、有经验、软实力和硬技能上对自己是自信的，也获得了面试官的认可
未来
1.个人希望在一家公司长期立足发展
2.个人希望倾注自己的时间去实现个人价值，为公司做出贡献，过低的薪资是很影响积极性的。
```

### 2.面对hr的否定

```
首先肯定自己，相信自己是有能力胜任的。
并且自己和市场上横向对比，都是比咱这儿高的

```

### 3.该提什么问题

1）如果说有股票、有晋升、或者有年终、正式编转正机会这种，那应该问比利是多少

2）问该部门研发的主要产品是什么？以及所面试职位在这个部门当中的作用

3）五险一金的比例

4）是否有股票和期权呢？

## 拒绝

查询

```
       1.非常感谢贵公司对我的认可。但是由于（什么理由），不能接受贵公司的offer，我觉得非常遗憾。希望我们可以保持联系，以后如果有机会，希望能与贵公司由更紧密的联系。但这次我真的非常抱歉，还请贵公司理解。谢谢。 

​       2.非常感谢贵公司愿意给我机会，并给我提供了适合我的职位。我非常珍惜这一次机会，但是由于（什么理由），没有办法成为公司的一员，我很遗憾。希望以后可以有合作的机会，还请贵公司理解。谢谢。 

​       3.非常感谢贵公司的青睐，之前也对贵公司非常的向往，但我发现贵公司提供的平台与我的职业发展方向有一定的偏离，没办法接受贵公司的offer，十分抱歉。但希望以后能有幸与贵公司合作，还请贵公司理解。谢谢。 

​       4.非常感谢贵公司发给我的offer邀请，这次机会对我来说十分珍贵，我也特别喜欢贵公司。但是由于工作地点，我思虑良久，还是说声抱歉，今日向您申请取消入职。给贵公司带来的不便还请谅解，希望贵公司能找到更加适合这个职位的人选，若今后有机会也非常希望能再次合作！真诚地说声抱歉！ 

​       5.接到贵公司发给我的offer邀请，我十分开心和荣幸。我思考很久，觉得自己在求职方面还有些事情考虑得不周全，思来想去还是决定向您申请放弃这次宝贵机会。特别抱歉给公司的工作带来不便，希望公司能有更好。 

​       最后，奉劝大家，不是迫不得已不要回绝人家的OFFER，毕竟这是一种缘分，而且你心仪的另一家公司未必有这家好~ O(∩_∩)O哈哈~~ 所以大家回绝的时候一定要谨慎、谨慎、再谨慎~


```
