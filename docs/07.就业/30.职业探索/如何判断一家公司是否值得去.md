如何判断一家公司是否值得去、值得长期发展

```
入职前：了解行业、基本薪资、福利待遇、晋升机制、公司产品是否确实造福社会、是否尊重个人发展、看融资轮数、看是谁投资他的、公司的核新产品是什么、在市场上的竞争力如何？
入职后：公司氛围、公司是否愿意分钱、看部门leader、
```
