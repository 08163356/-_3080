Vuex 和单纯的全局对象有以下两点不同：

```
1.VUEX的状态存储时响应式的。当VUE组件从store中读取状态的时候，若store中的状态发生变化，那么相应的组件也会相应地得到高效更新
2。你不嫩那个直接改变store中的状态，改变store中的状态的唯一途径就是显式地提交mutation
```
