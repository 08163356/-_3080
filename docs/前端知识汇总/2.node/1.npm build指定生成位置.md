vue webstorm build 打包 指定目录

本质是项目部署：https://www.codeleading.com/article/36155284215/



在vue.config.js中进行配置：

```js
module.exports = {
    productionSourceMap: false,
    publicPath: '../../../../../../../AndroidStudioProjects/hiBrowser/app/src/main/assets/',
    outputDir: 'settings',
    assetsDir: 'assets',
    devServer: {
        port: 8091,
        host: '0.0.0.0',
        https: false,
        open: true
    }
}
```
