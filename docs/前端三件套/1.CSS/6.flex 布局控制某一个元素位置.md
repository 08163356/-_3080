使用V-FOR产生了多个元素之后，需要对其中的某一个元素的位置进行重新设定。

使用

```
>span:last-child {
```

其中的某一个元素：

```
>span:nth-child(4) {
    -webkit-text-stroke: 3px #000;
}
```
