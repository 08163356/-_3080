value below is evaluated just now



chrome的控制台的`console.log()`展开的时候会去重新查询一次数组的值 

#### 如何避免呢

1. 使用`document.write()`直接显示到页面上
2. 使用`JSON.stringify()`转化为json字符串进行输出
3. 打印之前赋值给一个新的数组，注意要`改变引用地址`的那种才行
