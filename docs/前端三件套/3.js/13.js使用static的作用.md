把一个方法作为一个整体赋值给类。这样的方法被称为 **静态的（static）**。

通常，静态方法用于实现属于整个类，但不属于该类任何特定对象的函数。

对于变量：

在调用JavaScript函数之前，静态变量不会产生任何影响。静态变量是在函数完成执行后仍然保留其值的变量。它们是用关键字“static”声明的，并且只能在声明它们的函数中访问。静态变量通常用于存储需要在多个函数调用中使用的值。
