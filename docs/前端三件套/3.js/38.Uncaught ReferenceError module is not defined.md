报错位置

```
module.exports = {
    Association,
	Application
	//closeSettingPage
};
```

```
Uncaught ReferenceError: module is not defined
    at main.js:261
```

解决方法

```
try {
  console.log(window.name)
} catch (e) {
  module.exports = {
    Association,
	Application
	//closeSettingPage
};
}
```
