# 一、knowledge

1.浏览器进程之间的通信很常见

2.沙箱以进程为基本单位，防止浏览器在被有意或者无意、内部或者外部攻击时将访问区域限定在一个范围内，确保不会对整个系统产生影响。

3.tab间进行切换的时候

4.渲染进程是不能往系统写数据的，需要写数据则改成单进程，让render进程也在browser主进程中。
